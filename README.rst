Developing
----------

You can easily run this project under the Django development web server
by cloning this project: https://github.com/tgs/webhook-project-site


Running the Tests
------------------------------------

You can run the tests with via::

    python setup.py test

or::

    make test

or::

    make all

or::

    python runtests.py
